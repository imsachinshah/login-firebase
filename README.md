# login_firebase

A Flutter Login App intialized with Firebase

## Getting Started

An App to learn and implement Google Firebase with Flutter Application.

### Features

- Login
- SignUp
- HomeScreen
- LogOut

### Features to Come

- Push Notification
- More Firebase Features

- firebase_core: ^2.3.0
- fluttertoast: ^8.1.1
- firebase_messaging: ^14.2.1
- flutter_local_notifications: ^13.0.0
